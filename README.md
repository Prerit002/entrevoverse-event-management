# Entrevoverse Event Management System

A web application built for Entrevoverse startup to manage events and registrations.

## Features
- View upcoming events
- User registration for events
- Admin panel to add/delete events and view registrations

## Technologies Used
- Java 17
- Spring Boot
- REST API
- MySQL
- HTML, CSS, JavaScript

## How to Run
1. Clone the repository
2. Setup MySQL database named `entrevoverse`
3. Update `application.properties` with your MySQL credentials
4. Run `mvn spring-boot:run`
5. Open `http://localhost:8080`
